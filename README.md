## 🚀 Getting started

1. 進入 [最新 Release](https://github.com/Gholts/yaml/releases/latest) 下載對應版本即可。

2. 把配置檔案中的**機場連結**或**節點資訊**替換為你的。

3. 如果需要客製化配置檔案中的內容，請參考 [虛空終端](https://wiki.metacubex.one/config/) 使其為正確的格式。
